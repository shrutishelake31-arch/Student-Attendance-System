# Face Recognition Attendance System

A Python-based Face Recognition Attendance System developed using OpenCV and face recognition technology.  
The system automatically detects faces through webcam and marks attendance efficiently.


## Features

- Face detection using webcam
- Automatic attendance marking
- Stores attendance in CSV file
- Real-time face recognition
- Simple and user-friendly interface
- Date and time recording
- Reduces manual attendance work

## Technologies Used

- Python
- OpenCV
- Face Recognition
- CSV File Handling
- Tkinter GUI


## Applications

- Schools
- Colleges
- Offices
- Smart classrooms
- Employee attendance systems


## Advantages

- Fast attendance process
- Accurate face detection
- Saves time
- Reduces human errors
- Contactless attendance system


## Project Objective

The objective of this project is to develop an intelligent attendance management system using face recognition technology.  
The system automatically identifies students using webcam and records attendance digitally.
